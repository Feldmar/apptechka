# Аптечка

Веб-приложение для напоминаний о приёме лекарств с бэкендом, календарём приёмов и поддержкой экстренных препаратов без напоминаний.

## Возможности

- Добавление препаратов с напоминанием (название, дозировка, время)
- Экстренные препараты без уведомлений
- Кнопка «Принял» — запись приёма в историю
- Календарь с просмотром принятых препаратов по дням
- Хранение данных в SQLite на бэкенде

## Запуск

```bash
npm install
npm install --prefix server
npm run dev
```

- Фронтенд: http://localhost:5173
- API: http://localhost:3001

Разрешите уведомления в браузере для работы напоминаний.

## Сборка

```bash
npm run build
npm run build:server
```

## API

- `GET /api/medications` — список препаратов
- `POST /api/medications` — добавить препарат
- `DELETE /api/medications/:id` — удалить препарат
- `GET /api/intakes?from=&to=` — история приёмов
- `POST /api/intakes` — записать приём
- `DELETE /api/intakes/:id` — удалить запись

## Структура

```
server/          — Express + SQLite API
src/             — React + MUI фронтенд
```
